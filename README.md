# Acciona IT test

Aplicación para descubrir latitud y longitud de cada provincia argentina.

Datos obtenidos a travez de la API oficial del ministerio de modernización https://datosgobar.github.io/georef-ar-api/

## Instalación

1. Clone repo from https://github.com/juanem1/accionait.git
2. `cd accionait`
3. `npm install`
4. `ng serve`

## URLs

1. Login http://localhost:4200/ o http://localhost:4200/login
2. Search http://localhost:4200/search

## Files
Los archivos de la aplicacion estan locaclizados dentro de la carpeta /src/app.

Esta carpeta esta dividida en:
- Componentes /src/app/components
- Servicios /src/app/services
